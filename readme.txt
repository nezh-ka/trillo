1. ОТОБРАЗИТЬ КАКОЕ-ТО МЕНЮ ПОЛЬЗОВАТЕЛЯ ПРИ НАВЕДЕНИИ КУРСОРА НА ИМЯ ПОЛЬЗОВАТЕЛЯ В .user-nav;

2. ОТОБРАЗИТЬ МЕНЮ СООБЩЕНИЙ ПРИ НАВЕДЕНИИ КУРСОРА НА ЗНАЧОК ЧАТА В .user-nav (ВОЗМОЖНО, КАК FACEBOOK);

Итак, первая проблема, с которой я столкнулся, это

для вас, чтобы отобразить какое-то пользовательское меню, когда пользователь

парит над именем пользователя в user-nav.

Итак, что-то вроде пользовательских настроек

или что-то типа того.

Тогда что-то очень похожее, чтобы отобразить меню сообщений

при наведении курсора на значок чата в пользовательской навигации,

так что представьте что-то вроде Facebook,

поэтому, когда вы нажмете значок сообщения, то он показывает

список ваших последних сообщений, верно?

Таким образом, вы могли бы написать что-то подобное.

3. ОТОБРАЗИТЬ ОКНО С ПРЕДЛОЖЕНИЯМИ ПОИСКА, КАК ТОЛЬКО ПОЛЬЗОВАТЕЛЬ НАЧНЕТ ВВОДИТЬ ТЕКСТ В ПОЛЕ ПОИСКА;

Еще одна вещь, которая была бы действительно крутой, это

отобразить окно с предложениями, как только пользователь

начинает печатать в поле ввода,

так там в нашей панели поиска.

Так что представьте, когда вы что-то гуглите,

и он также показывает вам эту коробку с некоторыми предложениями,

и вы могли бы написать что-то подобное.

4. СОЗДАЙТЕ ЗАГОЛОВОК ДЛЯ .gallery__item С ХОРОШИМ ЭФФЕКТОМ НАВЕДЕНИЯ;

Далее, вы также можете создать подпись для галереи

предмет с хорошим эффектом наведения,

так что сейчас у нас есть только элемент фигуры,

с изображением, но без подписи,

и поэтому вы можете добавить одну подпись и показать, что

с хорошим эффектом наведения при наведении на изображение.
